Learn The Fundamentals

Frontend development stands firmly on three pillars: HTML for markup, CSS for styling and JavaScript for logic and interactions.

It is also closely related to web design and user experience, and the line between the two can be different in every company. Typically, frontend developers focus on technical issues, while web designers focus on aesthetics and UX.

if your looking to make a complex application, using it without React will result in masses of complex boilerplate code.
